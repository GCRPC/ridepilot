Runs

Runs refer to individual routes driven by a Vehicle. A single Run may include multiple Customer Trips.

The Runs tab displays a summary of the defined Runs based on the selected Filter, and allows you to create new Runs, and to edit and delete existing Runs.

The Runs page is divided into 3 areas:

1.      Filter panel on the left of the page allows you to filter the Runs displayed in the Calendar and Table views, based on a number of factors including date range and various Vehicle, Driver and Run attributes;

2.      Calendar view of the Runs, which includes a checkbox to hide the calendar from view or re-display it; and,

3.      Table view of the Runs.

Run Filters

To apply a filter:

1.      Enter the required filter parameters, and click OK at the top of the panel.

2.      The Calendar and Table views will be updated to display Runs that match the filter.

 

To reset the filter:

1.      Click Reset at the top of the panel.

2.      The Filter will be reset to the defaults, and the Calendar and Table views will be updated to display Runs that match the default filter.

Display a Run

To view the details of a Run, either:

·         Click the appropriate Run box in the Calendar view; or,

·         Click View next to the Run in the Table view.

Either way, the details of the selected Run will be displayed. To return to the Run Summary page, click Back.

New Run

To create a new Run:

3.      Click New Run at the top-right of the page.

4.      A blank Run details page is displayed:

5.      Enter all of the required information, and click Save.

6.      Assuming all of the entered Run information is validated, the Run will be created.

Edit Run

To edit an existing Run:

7.      Display the details of the existing Run.

8.      Click Edit at the top-right of the page.

9.      The Run details will be displayed in "Edit Mode" allowing the values to be edited:

10.  Make the appropriate edits, and click Update at the top-right of the page.

11.  Assuming all of the edits are, the Run will be updated.

Delete Run

To delete an existing Run:

1.      Display the details of the existing Run.

2.      Click Delete at the top-right of the page.

3.      A confirmation dialog is displayed asking you to confirm the deletion. Click OK to confirm the deletion.

4.      The Run will be deleted, and any assigned Trips will be unscheduled.

The Run is "soft" deleted meaning while it is not visible within the application, it still exists in the database. Contact a System Administrator if you mistakenly delete a Run and need it to be recovered.
