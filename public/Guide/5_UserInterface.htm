User Interface

The basic controls of the RidePilot application appear on each page:

路       Title bar containing the name of the application, the name of the User who is currently logged in, and the name of the selected Provider (typically the Provider that the User is affiliated with).

路       A series of tabs that display the various pages within the application. Each of these tabs is described below. 

路       Page body that changes based upon the selected tab.

路       Page footer indicating the version number.
