RidePilot Help

System Administrators only

RidePilot uses a number of functions, workflows and terms that may not be obvious or familiar to users, and so could benefit from additional explanation. To allow flexibility, RidePilot can provide the help:

·         As part of this Guide;

·         As pop-up context-sensitive help associated with each panel, accessible via the  in the top-right corner of the panel; and/or

·         As pop-up context-sensitive help associated with each displayed label, accessible via the  adjacent to the label.

Each of these help methods are described below.

RidePilot Guide

While this RidePilot Guide does cover the basic trip planning functionality within RidePilot, it is targeted at novice users as most of the functionality within the system should be self-explanatory.

As new functionality is implemented within RidePilot, this guide should be enhanced to incorporate references to this additional functionality.

Panel Help

Help can be associated with each panel displayed within RidePilot. Panel help is available if there is a  in the top-right corner of the panel. The help is displayed in a pop-up window by hovering the mouse pointer over the .

The panel help is maintained by the System Administrator using the RidePilot Translations functionality.

To view the panel help source:

1.      Open RidePilot and login as a System Administrator.

2.      Determine for the panel title translation key. The translation keys can be displayed by clicking Tags in the bottom-right of the page. Once determined, revert to the appropriate language.

3.      Display the Admin tab and click Translations to display the Translations page.

4.      Search for the determined panel title translation key. If there is help associated with the panel, there will be another translation key postfixed with _help. For example, the translation key for the Trip Parameters panel on the Trip page is [trip_parameters] so the associate help translation key would be [trip_parameters_help].

If the help translation key does not exist, it is assumed there is no help associated with the panel.

To edit the panel help source:

1.      Either:

§  If the help translation key does not exist, click Add Key to create a new key.

§  If the help translation key exists but there is no help defined, click Add for the translation in the appropriate language column.

§  If the help translation key exists and help is defined, click the hyperlink for the translation in the appropriate language column.

2.      A page is displayed allowing you to enter/edit the help text. Note: If you created a new key, make sure the key name is set correctly with the postfix _help.

3.      Click Save to save the help text.
