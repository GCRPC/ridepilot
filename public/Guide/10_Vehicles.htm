Vehicles

The details of each Provider Vehicle can be entered and maintained, including maintenance  and compliance events.

The Vehicles tab displays a list of the defined Vehicles, and allows you to create new Vehicles, and update existing Vehicles.

Display a Vehicle

To view the details of a Driver:

1.      Click the Vehicle name hyperlink in the table.

2.      The details of the selected Vehicle will be displayed. To return to the Vehicle summary page, click Back.

New Vehicle

To create a new Vehicle:

1.      Click either New Vehicle at the top-right of the page.

2.      A blank Vehicle details page is displayed:

3.      Vehicle fields include Name, Year, Make, Model, License Plate, VIN, Driver, Active (yes/no), inclusion in Vehicle Report, Initial Mileage, Seating Capacity, and Mobility Devices Accommodated. Additional fields may be set under the Additional Information section as well.

4.      Enter all of the required information, and click Save.

5.      Assuming all of the entered Vehicle information is validated, the Vehicle will be created.

Edit Vehicle

To edit an existing Vehicle:

1.      View the details of the existing Vehicle.

2.      Click Edit at the top-right of the page.

3.      The Vehicle details will be displayed in "Edit Mode" allowing the values to be edited. All of the vehicle fields listed above in the New Vehicle section may be edited, including Initial Mileage.

4.      Make the appropriate edits, and click Update at the top-right of the page.

5.      Assuming all of the edits are validated, the Vehicle will be updated.

Vehicle Documentation

Documents related to the Vehicle can be uploaded into the system. These documents can then be related to Maintenance and Compliance Events.

View a Vehicle Document

To view any of the uploaded Vehicle documents:

1.      View the details of the existing Vehicle.

2.      In the “Vehicle Documentation” panel, click the Description hyperlink in a document row.

3.      The selected document is downloaded to your computer in preparation for opening.

Add a Vehicle Document

To upload a document for a Vehicle:

1.      View the details of the existing Vehicle.

2.      Click Add document at the top-right of the page.

3.      An Upload new document page, similar to the page for uploading a driver document, is displayed.

4.      Enter a description for the document, and select Choose File to locate and select a file to be uploaded.

5.      Click Save to upload the document.

Edit a Vehicle Document

To replace a previously uploaded document for a Vehicle with an updated version of the document:

1.      View the details of the existing Vehicle.

2.      In the “Vehicle Documentation” panel, click Edit in a document row.

3.      The Document will be displayed in "Edit Mode" allowing the values to be edited.

4.      Enter a description for the document, and select Choose File to locate and select an updated version of the document to be loaded.

5.      Click Save to update the Document.

Delete a Vehicle Document

To delete a previously uploaded document for a Vehicle:

1.      View the details of the existing Vehicle.

2.      Click Edit at the top-right of the page.

3.      The Vehicle details will be displayed in "Edit Mode" allowing the values to be edited.

4.      In the Vehicle Documentation panel, click X next to the document to delete.

5.      A dialog is displayed to confirm the deletion. Click OK to delete the document.

6.      Click Save to update the Vehicle.

Vehicle Events and Warranties

To maintain a historical record regarding the warranties, maintenance and compliance of each Vehicle, Warranties, and Maintenance and Compliance Events can be entered.

Existing events and warranties are displayed in the “Maintenance Events”, “Compliance Events” and “Warranty” panels.

Add a Vehicle Event or Warranty

To add a new Maintenance or Compliance Event, or a Warranty for a Vehicle.

1.      View the details of the existing Vehicle.

2.      Click Edit at the top-right of the page.

3.      The Vehicle details will be displayed in "Edit Mode" allowing the values to be edited.

4.      In the Maintenance Events, Compliance Events or Warranty panel, as appropriate, click Add Event. A dialog is displayed allowing the details of the event or warranty to be entered:

  

5.      Enter the required information for the event or warranty.

6.      If there are any documents that should be associated with the event or warranty, select them from the Document dropdown list.

7.      Click Save to add the event or warranty.

8.      Click Save to update the Vehicle.

Note: To add a recurring Vehicle Maintenance Compliance Event that will be applied to all vehicles on a recurring basis, go to the Admin tab and click on the Recurring Vehicle Maintenance Compliance Eventslink.

Delete a Vehicle Event or Warranty

To delete an existing event or warranty for a Vehicle:

1.      View the details of the existing Vehicle.

2.      Click Edit at the top-right of the page.

3.      The Vehicle details will be displayed in "Edit Mode" allowing the values to be edited.

4.      In the Maintenance Events, Compliance Events or Warranty panels, click X next to the event to delete.

5.      A dialog is displayed to confirm the deletion. Click OK to delete the event.

6.      Click Save to update the Vehicle.

Delete a Vehicle

To delete an existing Vehicle:

1.      View the details of the Vehicle.

2.      Click Delete at the top-right of the page.
