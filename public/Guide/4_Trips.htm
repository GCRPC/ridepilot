Trips

A Trip defines the details of the trip request, including the start and end locations, the pickup and drop-off times, eligibilities, accommodations, and the size of the group that is traveling.

The Trips tab displays a summary of the entered Trips based on the selected Filter, and allows you to create new Trips, and to edit and delete existing Trips.

The Trips page is divided into 3 areas:

1.      Filter panel on the left of the page allows you to filter the Trips displayed in the Calendar and Table views, based on a date range and various Vehicle, Driver and Trip attributes;

2.      Calendar view of the Trips, which includes a checkbox to hide the calendar from view or re-display it; and,

3.      Table view of the Trips.

Trip Filters

Filters can be applied to the Trips to only display Trips that match given criteria (e.g., with a date range, completed only, etc.)

To apply a filter:

1.      Enter the required filter parameters, and click OK at the top of the panel.

2.      The Calendar and Table views will be updated to display Trips that match the filter.

 

To reset the filter:

1.      Click Reset at the top of the panel.

2.      The Filter will be reset to the defaults, and the Calendar and Table views will be updated to display Trips that match the default filter.

Display a Trip

To view the details of a Trip, either:

1.      Click the appropriate Trip box in the Calendar view; or,

2.      Click View next to the Trip in the Table view.

To return to the Trips summary page, click Back.

New Trip

To create a new Trip:

1.      Click New Trip at the top-right of the page.

2.      A blank Trip details page is displayed:

3.      Enter all of the required information, and click Save.

4.      Assuming all of the entered Trip information is validated, the Trip will be created.

5.      A confirmation dialog will show up on whether to create a Return Trip.

6.      Click No to skip creating a Return Trip, or click Yes to get into Return Trip editing mode.

7.      Enter all of the required information, and click Save. The Return Trip will be created.

Although the Trip has been created, it will not be assigned to a Run.

Edit Trip

To edit an existing Trip:

1.      Display the details of the existing Trip.

2.      Click Edit at the top-right of the page.

3.      The Trip details will be displayed in "Edit Mode" allowing the values to be edited:

4.      Make the appropriate edits, and click Update at the top-right of the page.

5.      Assuming all of the edits are validated, the Trip will be updated.

Clone Trip

To clone an existing Trip:

1.      Display the details of the existing Trip.

2.      Click Clone at the top-right of the page.

3.      The Trip details will be displayed in "Clone Mode" allowing the values to be edited.

4.      Make the appropriate edits, and click Save at the top-right of the page.

5.      Assuming all of the edits are validated, a cloned Trip will be created.

Delete Trip

To delete an existing Trip:

1.      On the Trips page, click on the Trip you wish to deleted in order to display its details.

2.      Click Delete at the top-right of the page.

3.      A confirmation dialog is displayed asking you to confirm the deletion. Click OK to confirm the deletion.

4.      The Trip will be deleted, and removed from any Runs.

The Trip is "soft" deleted meaning while it is not visible within the application, it still exists in the database. Contact a System Administrator if you mistakenly delete a Trip and need it to be recovered.
