Trips-Runs

The Trips-Runs tab allows Trips to be assigned to Runs.

While the Trips and Runs tabs use date ranges as part of the Filter, the Trips-Runs tab focuses on a specific day.

The Trips-Runs summary page is divided into 3 areas:

1.      Filter panel on the left of the page allows you to filter the Trips and Runs displayed in the Hour and Table views, based on various factors including date range and various Vehicle, Driver and Run attributes;

2.      Hourly view broken down by defined Runs, Cab, and Unscheduled Trips; and,

3.      Table view of the Trips for the selected day.

Trips-Runs Filter

To apply a filter:

1.      Enter the required filter parameters, and click OK at the top of the panel.

2.      The Hourly and Table views will be updated to display Trips and Runs that match the filter.

 

To reset the filter:

1.      Click Reset at the top of the panel.

2.      The Filter will be reset to the defaults, and the Calendar and Table views will be updated to display Runs that match the default filter.

Scheduling Trips

To schedule an existing Trip (i.e., assign the Trip to an existing Run), either:

1.      Select the appropriate Run from the 'Status' dropdown list for the Trip in the Table view.

Or:

1.      Select the appropriate Trip in the Table view by selecting the radio button to the left of the Trip.

2.      Select the appropriate Run in the Hourly view by selecting the radio button to the left of the Run.

3.      Click Schedule to schedule the Trip.

Unscheduling Trips

To unschedule a Trip that has been assigned to a Run, either:

1.      Select "Unscheduled" from the 'Status' dropdown list for the Trip in the Table view.

Or:

1.      Select the appropriate Trip in the Table view by selecting the radio button to the left of the Trip.

2.      Click Unschedule to unschedule the Trip.
