Drivers

Each Driver needs to be defined to capture their contact details, history and compliance.

The Drivers tab displays a list of the defined Drivers, and allows you to create new Drivers, and update existing Drivers.

Display a Driver

To view the details of a Driver:

1.      Click the Driver name hyperlink in the table.

2.      The details of the selected Driver will be displayed. To return to the Driver summary page, click Back.

New Driver

To create a new Driver:

1.      Click either New Driver at the top-right of the page.

2.      A blank Driver details page is displayed:

3.      Enter all of the required information, and click Save.

4.      Assuming all of the entered Driver information is validated, the Driver will be created.

Edit Driver

To edit an existing Driver:

1.      View the details of the existing Driver.

2.      Click Edit at the top-right of the page.

3.      The Driver details will be displayed in "Edit Mode" allowing the values to be edited.

4.      Make the appropriate edits, and click Update at the top-right of the page.

5.      Assuming all of the edits are validated, the Driver will be updated.

Driver Documentation

Documents related to the Driver can be uploaded into the system. These documents can then be associated with History and Compliance Events that need to be tracked for the Driver.

View a Driver Document

To view any of the uploaded Driver documents:

1.      View the details of the existing Driver.

2.      In the “Driver Documentation” panel, click the hyperlink in the Description column for the document that you wish to view.

3.      The selected document is downloaded to your computer in preparation for opening.

Add a Driver Document

To upload a document for a Driver:

1.      View the details of the existing Driver.

2.      Click the Add document button at the top-right of the page.

3.      A Upload new document page is displayed allowing the document to be selected:

4.      Enter a description for the document, and select Choose File to select the file to upload.

5.      Click Save to upload the document.

Edit a Driver Document

To updated a previously uploaded document for a Driver with a new version of the document:

1.      View the details of the existing Driver.

2.      In the “Driver Documentation” panel, click Edit in a document row.

3.      The Document will be displayed in "Edit Mode" allowing a new document file to be uploaded.

4.      Enter a description for the document, and select Choose File to locate and upload the file.

5.      Click Save to update the Document.

Delete a Driver Document

To delete a previously uploaded document for a Driver:

6.      View the details of the existing Driver.

7.      Click Edit at the top-right of the page.

8.      The Driver details will be displayed in "Edit Mode" allowing the values to be edited.

9.      In the Driver Documentation panel, click X next to the document to delete.

10.  A dialog is displayed to confirm the deletion. Click OK to delete the document.

11.  Click Save to update the Driver.

Driver Events

To maintain a historical record regarding each Driver and to log compliance-related actions, History and Compliance Events can be entered.

Existing events are displayed in the History Events or Compliance Events panels.

Add a Driver Event

To add a new History or Compliance Event for a Driver:

1.      View the details of the existing Driver.

2.      Click Edit at the top-right of the page.

3.      The Driver details will be displayed in "Edit Mode" allowing the values to be edited.

4.      In the History Events or Compliance Events panel, as appropriate, click Add Event. A dialog is displayed allowing the details of the event to be entered:

 

5.      Enter the name, notes and date(s) for the event.

6.      If there are any documents that should be associated with the event, select them from the Document dropdown list.

7.      Click Save to add the event.

8.      Click Save to update the Driver.

Note: To add a recurring Driver Compliance Event that will apply to all drivers on a recurring basis, go to the Admin tab and select the Recurring Driver Compliance Events link.

Delete a Driver Event

To delete an existing event for a Driver:

1.      View the details of the existing Driver.

2.      Click Edit at the top-right of the page.

3.      The Driver details will be displayed in "Edit Mode" allowing the values to be edited.

4.      In the History Events or Compliance Events panel, click X next to the event to delete.

5.      A dialog is displayed to confirm the deletion. Click OK to delete the event.

6.      Click Save to update the Driver.

Delete a Driver

To delete an existing Driver:

1.      View the details of the Driver.

2.      Click Delete at the top-right of the page.
