Reports

Reports can be run based on virtually any of the data maintained within the database.

To run a Report:

1.      Select the Reports tab.

2.      Select the required Report from the dropdown box at the top of the page.

3.      Based on the selected report, various filter panels and fields are displayed. To open a different panel, click the name of the panel.

4.      Enter the required filter parameters.

5.      Click Run to execute the report.

6.      The executed report is displayed in a table.

 

Many of the standard reports (i.e., those with less formatting) can be exported to a Comma Separated Value (CSV) file that can be opened by third-party applications such as Microsoft Excel. To export a Report to CSV:

1.      Click Export.

2.      The CSV file is created and downloaded to your computer in preparation for opening.
